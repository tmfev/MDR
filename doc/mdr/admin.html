Server Administration

Update

Dedicated update scripts ship with every release of the server.

To run a specific update, start the server and initiate the script using curl or a browser.

The update servlet returns HTTP 200 (OK) on success. It is recommended to check the server log files after the update to verify everything is ok.

The following update scripts exist:

update/update-0.8.26-0.8.34.sql: Updates the database from schema version 0.8.26 to 0.8.34.
