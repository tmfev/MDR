Architecture

The Metadata Repository is a browser-based 3-tier application. It is based on an SQL database (currently Apache Derby). On top of Derby runs Hibernate as an ORM which implements the domain model. The project is restricted to the JPA implementation of Hibernate. Check the list of dependencies for a complete set of all third party libraries.

There is a JSON Restful API for both browser and automation clients available. This API is hosted as a servlet that runs commands in the business layer.

The browser runs a GWT application and features a partial and simplified model of the domain model.

Data Interfaces

The MDR features 2 distinct interfaces to bring data into and out of the server. Meta Data Templates map XML data to the MDR domain model and help you to integrate external data present as XML. The MDR Rest API defines a remote API to fetch content at runtime by making HTTP JSON calls.
