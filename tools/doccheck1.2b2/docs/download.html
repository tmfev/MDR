Sun Doc Check Utilities


The Sun Doc Check utilities consist of a doc check doclet, PackageList utility, and pathname pattern-matching classes. The doc check doclet extends the Javadoc tool to review JavaDoc comments in the source code, identifying gaps and irregularities. It produces an HTML report that seperates serious problems from trivial ones, and makes "suggestions" in the form of templates that show you what is needed. (To save time, those suggestions can then be copied into the code and edited.)

This bundle contains a README file, documentation, source code and class files, sample output and the source code that generated them.

Documentation - View the documentation online from the Sun Doc Check Home Page.

Installation - Uncompress the downloaded bundle in any directory you want. This process creates a directory called doccheck containing all the files.

Requirements - The DocCheck doclet requires version 1.2 (or later) of the JavaTM 2 SDK, Standard Edition.
